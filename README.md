# 俺の解決策『デザイナーAI』作戦

- 特定の分野においてのみパフォーマンスの高い、偏ったAIを作成させる。デザイナーが行うのはメモリ管理やディスク管理、実行許可のみで、あとは放置。
- デザイナーAIにかかる淘汰圧は「突然現れた課題に対して、その専門家を作成し、正答させる」という特殊なものになる。これなら劣化はしにくいはず。
- ただし、局所的な最適解を求めるのは、探索空間が広すぎて、成長スピードに限界がやってくる。原理的にか確率的にかはさておき、"天井"は低い。
- 既知の、もっとも精度の良いヒューリスティックが"ゲージ原理"なのだから、たぶん真に万能なAIのバイナリは、あれよりもずっと酷い見た目だ。
